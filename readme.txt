Helpr Agent helps agencies to improve their customer service and to increase their sales through their WordPress projects.

== Description ==

**Just one click away to get the proper assistance for your WordPress website.**

<strong>Helpr Agent</strong> was built to keep in touch with your developer. 
If you or your customers needs help while working with Wordpress, this plugin will be there to send the request. Right there. Just type your issue or your questions and the emails are sent straight from your Wordpress dashboard. No call, no text, no complicated forms, nu other windows opened. <strong>Just one click</strong>. 
That's help desk at it's best.

**The request gets to you within seconds.**

If you are a developer you can add this plugin and let the owner of the website to run his business. Every time when he will need help, Helpr Agent will sent the emails and you can reply back.
<strong>It can be a useful tool for all kind of persons, developers or not</strong>.

== Installation ==

1. Install the plugin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin in Settings >Helpr Agent (you will be prompted to that page)
4. Press the "Request Help" tab from WordPress Dashboard
5. Send the request
